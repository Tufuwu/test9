Frequently Asked Questions

What is pastebin?

This pastebin is here to help you collaborate on debugging code snippets. If you're not familiar with the idea, most people use it like this:

  • submit a code fragment to pastebin, getting a url like {{ request.scheme }}://{{ request.get_host }}{% url 'snippet_details' 'abc1234' %}
  • paste the url into an IRC or IM conversation
  • someone responds by reading and perhaps submitting a modification of your code

How can I delete a post?

Your posted snippets are remembered in your session, you will be able to delete the snippet from the same browser you posted from - simply view the post and click the "delete" link.

In other cases, contact us and we will delete it for you.
